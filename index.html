Through this project, I demonstrated skills in game logic, user interface design, and event-driven programming in C#.

Automated Parking System – Arduino Project

Developed an automated parking system using an Arduino Due, ultrasonic sensors, servo motors, LEDs, an LCD display, and a traffic light system. The project simulates a smart parking lot with an automated lift mechanism for parking cars. Ultrasonic sensors detect available parking spaces, while servo motors control barriers to allow or block entry. The traffic light system manages the flow of vehicles, ensuring safe and efficient movement. An LCD display shows the number of available spaces, providing real-time information to users. This project showcases skills in integrating hardware components, programming for automation, and problem-solving in an embedded systems environment.

Square Wave Signal Generator – Arduino Project

Created a Square Wave Signal Generator using an Arduino platform to generate adjustable square wave signals for various applications. The project allows users to modify the frequency and duty cycle of the signal through an easy-to-use interface. Pulse Width Modulation (PWM) was utilized to control the output signal, providing precise and customizable square wave outputs. This project demonstrates a strong understanding of signal generation, microcontroller programming, and electronics, showcasing my ability to work with both hardware and software to create functional and adaptable systems.
